# 은행회계해설 AI 서비스 — 배포 가이드

## 1. 배포 방식 개요

Python/Ollama 설치 자동화 스크립트와 함께 ZIP으로 배포하는 방식을 사용한다.

```
은행회계해설AI/
├── install.bat              ← 더블클릭: 환경 설치
├── start.bat                ← 더블클릭: 서버 시작 + 브라우저 열기
├── src/                     ← 소스코드
├── templates/               ← 웹 UI
├── static/                  ← 스타일시트
├── docs/md/                 ← 마크다운 문서
├── chroma_db/               ← 벡터 DB (사전 구축본 포함)
├── requirements.txt         ← Python 패키지 목록
├── .env.example             ← API 키 템플릿
└── README.txt               ← 간단 사용 설명서
```

## 2. 배포 ZIP 만들기

### 2.1 포함할 파일

```
banking_accounting_rag/
├── src/
│   ├── app.py
│   ├── config.py
│   ├── rag_chain.py
│   ├── document_loader.py
│   └── build_vectordb.py
├── templates/
│   └── chat.html
├── static/
│   └── style.css
├── docs/
│   └── md/
│       ├── 은행회계해설(상)_2018개정판.md
│       └── 은행회계해설(하)_2018개정판.md
├── chroma_db/                ← 사전 구축된 벡터 DB (50분 절약)
├── requirements.txt
├── .env.example
├── install.bat
├── start.bat
└── README.txt
```

### 2.2 제외할 파일

| 파일 | 이유 |
|------|------|
| `.env` | API 키 포함 — 새 PC에서 직접 입력 |
| `docs/pdf/` | 원본 PDF, MD가 있으면 불필요 (용량 절감) |
| `__pycache__/` | Python 캐시, 자동 생성됨 |
| `.git/` | git 히스토리 불필요 |
| `구축계획서.md`, `구축완료보고서.md` | 개발 문서, 배포 불필요 |

### 2.3 ZIP 생성 명령

```powershell
# PowerShell에서 실행
cd C:\SIY_DEV\banking_accounting_rag

# 배포용 폴더 생성
$dest = "C:\SIY_DEV\은행회계해설AI_배포"
New-Item -ItemType Directory -Force -Path $dest

# 필요한 파일 복사
Copy-Item -Recurse src $dest\src
Copy-Item -Recurse templates $dest\templates
Copy-Item -Recurse static $dest\static
Copy-Item -Recurse docs\md $dest\docs\md
Copy-Item -Recurse chroma_db $dest\chroma_db
Copy-Item requirements.txt $dest\
Copy-Item .env.example $dest\
Copy-Item install.bat $dest\
Copy-Item start.bat $dest\
Copy-Item README.txt $dest\

# __pycache__ 제거
Get-ChildItem $dest -Recurse -Directory -Filter __pycache__ | Remove-Item -Recurse -Force

# ZIP 압축
Compress-Archive -Path $dest\* -DestinationPath "C:\SIY_DEV\은행회계해설AI.zip" -Force

# 임시 폴더 삭제
Remove-Item -Recurse -Force $dest
```

## 3. install.bat 작성

```bat
@echo off
chcp 65001 >nul
echo ===================================================
echo   은행회계해설 AI 서비스 — 설치 스크립트
echo ===================================================
echo.

:: 1. Python 확인
echo [1/4] Python 확인 중...
python --version >nul 2>&1
if %errorlevel% neq 0 (
    echo [오류] Python이 설치되어 있지 않습니다.
    echo        https://www.python.org/downloads/ 에서 Python 3.10 이상을 설치하세요.
    echo        설치 시 "Add Python to PATH" 체크 필수!
    pause
    exit /b 1
)
python --version
echo        OK
echo.

:: 2. Ollama 확인
echo [2/4] Ollama 확인 중...
ollama --version >nul 2>&1
if %errorlevel% neq 0 (
    echo [오류] Ollama가 설치되어 있지 않습니다.
    echo        https://ollama.com 에서 Ollama를 설치하세요.
    pause
    exit /b 1
)
ollama --version
echo        OK
echo.

:: 3. Python 패키지 설치
echo [3/4] Python 패키지 설치 중...
pip install -r requirements.txt
if %errorlevel% neq 0 (
    echo [오류] 패키지 설치 실패
    pause
    exit /b 1
)
echo        OK
echo.

:: 4. 임베딩 모델 다운로드
echo [4/4] 임베딩 모델(bge-m3) 다운로드 중... (약 1.2GB)
ollama pull bge-m3
if %errorlevel% neq 0 (
    echo [오류] 모델 다운로드 실패. Ollama 서버가 실행 중인지 확인하세요.
    pause
    exit /b 1
)
echo        OK
echo.

:: 5. .env 파일 생성
if not exist .env (
    copy .env.example .env >nul
    echo [참고] .env 파일이 생성되었습니다.
    echo        Gemini/OpenAI API를 사용하려면 .env 파일에 API 키를 입력하세요.
    echo.
)

:: 6. 벡터 DB 확인
if exist chroma_db\chroma.sqlite3 (
    echo [참고] 벡터 DB가 이미 포함되어 있어 재구축 불필요합니다.
) else (
    echo [참고] 벡터 DB가 없습니다. 최초 실행 시 구축이 필요합니다.
    echo        start.bat 실행 시 자동으로 안내됩니다.
)
echo.

:: 7. 로컬 LLM 안내
echo ===================================================
echo   설치 완료!
echo ===================================================
echo.
echo   [선택] 로컬 LLM 모드를 사용하려면:
echo          ollama pull gemma3:12b  (약 8.1GB 다운로드)
echo.
echo   [시작] start.bat 을 더블클릭하여 서비스를 시작하세요.
echo ===================================================
pause
```

## 4. start.bat 작성

```bat
@echo off
chcp 65001 >nul
echo ===================================================
echo   은행회계해설 AI 서비스 시작
echo ===================================================
echo.

:: 1. Ollama 서버 시작 (백그라운드)
echo [1/3] Ollama 서버 시작 중...
tasklist /FI "IMAGENAME eq ollama.exe" 2>nul | find /I "ollama.exe" >nul
if %errorlevel% neq 0 (
    start /B ollama serve >nul 2>&1
    timeout /t 3 /nobreak >nul
)
echo        OK
echo.

:: 2. 벡터 DB 확인
if not exist chroma_db\chroma.sqlite3 (
    echo [2/3] 벡터 DB 구축 중... (약 50분 소요)
    cd src
    python build_vectordb.py
    cd ..
    echo        OK
) else (
    echo [2/3] 벡터 DB 확인... OK
)
echo.

:: 3. Flask 서버 시작 + 브라우저 열기
echo [3/3] 웹 서버 시작 중...
echo.
echo   브라우저에서 http://localhost:5000 으로 접속하세요.
echo   종료하려면 이 창을 닫으세요.
echo ===================================================
echo.

start http://localhost:5000
cd src
python app.py
```

## 6. PyInstaller로 EXE 번들링 (선택)

Python 설치 없이 실행 가능한 EXE를 만들려면:

### 6.1 PyInstaller 설치

```bash
pip install pyinstaller
```

### 6.2 spec 파일 작성

`build_exe.spec`:
```python
# -*- mode: python ; coding: utf-8 -*-
import os

base_dir = os.path.dirname(os.path.abspath(SPEC))

a = Analysis(
    ['src/app.py'],
    pathex=[os.path.join(base_dir, 'src')],
    datas=[
        ('templates', 'templates'),
        ('static', 'static'),
        ('docs/md', 'docs/md'),
    ],
    hiddenimports=[
        'chromadb',
        'ollama',
        'google.genai',
        'openai',
        'flask',
        'langchain_core',
        'langchain_text_splitters',
    ],
)

pyz = PYZ(a.pure)

exe = EXE(
    pyz,
    a.scripts,
    [],
    name='은행회계해설AI',
    console=True,
    icon=None,
)

coll = COLLECT(
    exe,
    a.binaries,
    a.datas,
    name='은행회계해설AI',
)
```

### 6.3 빌드

```bash
pyinstaller build_exe.spec --distpath dist
```

### 6.4 결과

```
dist/은행회계해설AI/
├── 은행회계해설AI.exe     ← 이것만 실행하면 됨
├── (번들된 Python + 패키지)
└── (templates, static, docs 포함)
```

**주의:** PyInstaller로 번들해도 Ollama는 별도 설치 필요. ChromaDB의 네이티브 바이너리(hnswlib)가 번들 시 문제를 일으킬 수 있으므로 테스트 필수.

## 7. Inno Setup으로 설치 프로그램 만들기 (선택)

PyInstaller 결과물을 Inno Setup으로 감싸면 정식 Windows 설치 프로그램을 만들 수 있다.

### 7.1 Inno Setup 설치

https://jrsoftware.org/isinfo.php 에서 다운로드

### 7.2 스크립트 작성 (`setup.iss`)

```iss
[Setup]
AppName=은행회계해설 AI
AppVersion=1.0
DefaultDirName={autopf}\은행회계해설AI
DefaultGroupName=은행회계해설 AI
OutputBaseFilename=은행회계해설AI_Setup
Compression=lzma2
SolidCompression=yes

[Files]
Source: "dist\은행회계해설AI\*"; DestDir: "{app}"; Flags: recursesubdirs
Source: "chroma_db\*"; DestDir: "{app}\chroma_db"; Flags: recursesubdirs
Source: ".env.example"; DestDir: "{app}"; DestName: ".env"

[Icons]
Name: "{group}\은행회계해설 AI"; Filename: "{app}\은행회계해설AI.exe"
Name: "{commondesktop}\은행회계해설 AI"; Filename: "{app}\은행회계해설AI.exe"

[Run]
Filename: "{app}\은행회계해설AI.exe"; Description: "은행회계해설 AI 실행"; Flags: postinstall nowait
```

### 7.3 빌드

Inno Setup Compiler에서 `setup.iss`를 열고 Compile 클릭.

결과: `은행회계해설AI_Setup.exe` (약 200~300MB)

## 8. 배포 방식 비교

| 방식 | 난이도 | 사용자 편의성 | Python 설치 | Ollama 설치 | 크기 |
|------|--------|-------------|------------|------------|------|
| **ZIP + bat 스크립트** | 낮음 | 보통 | 필요 | 필요 | ~20MB |
| **PyInstaller EXE** | 중간 | 좋음 | 불필요 | 필요 | ~300MB |
| **Inno Setup 설치 프로그램** | 높음 | 매우 좋음 | 불필요 | 필요 | ~300MB |

**추천:** 사내 배포라면 **ZIP + bat 스크립트**가 가장 현실적. Python과 Ollama가 이미 있는 환경이라면 ZIP만으로 충분하다. 불특정 다수에게 배포한다면 PyInstaller + Inno Setup을 고려.
