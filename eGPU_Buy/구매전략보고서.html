1.2 로컬 LLM의 핵심 병목 요소

💡 핵심 인사이트: VRAM이 가장 중요하다
로컬 LLM 추론에서 성능을 결정하는 요소는 ① VRAM 용량 > ② 메모리 대역폭 > ③ 연산력(TFLOPS) 순이다. 모델 가중치를 GPU 메모리에 전부 올릴 수 있어야 실용적인 속도가 나온다.
병목 요소설명현재 노트북 상태
VRAM모델 가중치를 올리는 GPU 전용 메모리내장 GPU 공유 메모리뿐 → 부족
메모리 대역폭GPU ↔ VRAM 간 데이터 전송 속도내장 GPU ~89GB/s → 제한적
연산력행렬 연산 처리 능력 (TFLOPS)내장 GPU ~8 TFLOPS → 부족
시스템 RAMCPU 오프로드 시 사용16GB 고정, 증설 불가 → 여유 없음

Chapter 04

eGPU vs 미니PC 장단점 심층 비교

4.1 eGPU 방안 (AOOSTAR AG03)

👍 장점

  • 기존 노트북 활용 → GPU에 예산 집중
  • AG03 = PSU 내장 + 듀얼 TB5 + 140W PD → 케이블 1개 올인원
  • 외출 시 분리하면 노트북 이동성 유지
  • 기존 디스플레이/키보드/트랙패드 활용
  • LLM 추론에서 USB4 병목 제한적
  • ~30만 원으로 합리적인 독 가격

👎 단점

  • TB4 하위호환 시 40Gbps → prefill 20~30% 하락
  • 노트북 RAM 16GB 고정 → CPU 오프로드 여유 부족
  • 오픈 프레임 → 먼지, GPU 팬 소음 노출
  • 중고 GPU 사용 시 채굴/보증 리스크
  • 드라이버/절전모드 호환성 이슈 가능
  • 해외직구 필요 (국내 정식 유통 없음)

4.2 미니PC 방안

👍 장점

  • 독립 LLM 전용 서버 (24시간 상시 가동)
  • OCuLink → USB4보다 높은 대역폭
  • 32GB RAM 구성 → CPU 오프로드 여유
  • SSH/API 원격 접속 가능
  • 노트북 배터리/성능 부담 없음
  • 소형 폼팩터

👎 단점

  • 본체 55~90만 원 → 예산 초과 가능
  • 예산 내 RTX 3060 12GB 수준 → 13B까지만
  • 별도 모니터/키보드 필요
  • 이미 노트북 있는데 PC 추가 → 비용 비효율
  • 2대 관리 부담
  • 대형 GPU 물리적 호환 불가 가능

B안 추천 이유:

  • 신품 GPU → 제조사 보증, 채굴 이력·고장 리스크 없음
  • Blackwell 아키텍처 + GDDR7 → 4060 Ti 대비 15% 성능 향상, 전력 효율 우수
  • 16GB VRAM으로 Gemma 3 12B, Qwen 14B를 쾌적하게 구동
  • AOOSTAR AG03의 800W PSU + 140W PD → 케이블 1개로 GPU + 충전 동시
  • AG03 듀얼 TB5 → 차세대 TB5 노트북 업그레이드 시 대역폭 80Gbps로 2배 향상
  • 예산 100~150만 원 범위에 정확히 맞는 안정적 구성
