# 로컬 LLM 구동을 위한 eGPU vs 미니PC 구매전략보고서 작성 계획

## 1. 보고서 목적

현재 사용 중인 노트북에서 로컬 LLM을 구동하기 위한 최적의 하드웨어 확장 방안을 비교 분석하고, 예산 100만~150만 원 내에서 최선의 구매 전략을 도출한다.

## 4. 보고서 구성 (목차)

### 제1장. 현황 분석
- 현재 노트북 사양 및 eGPU 호환성 (TB4/USB4 x 2포트 확인)
- 내장 Intel Arc GPU의 LLM 구동 한계
- 로컬 LLM 구동 시 핵심 병목 요소 (VRAM, 메모리 대역폭, 연산력)
- 타겟 모델 규모 설정 (7B / 13B / 30B 파라미터)

### 제2장. 방안별 비교 분석

#### 2-1. eGPU 방안 (Galaxy Book4 Pro + AOOSTAR AG03 + GPU)
- AOOSTAR AG03 eGPU 독 상세 사양 및 가격 (~30만 원)
  - 800W PSU 내장 → 별도 파워 불필요
  - USB4 포트로 Galaxy Book4 Pro TB4 연결 (40Gbps)
  - 100W PD 충전 → 케이블 1개로 GPU + 충전 동시
- GPU 카드 선택지 및 가격 (2026.04 기준 실제 시장가)
  - RTX 3090 중고 (24GB VRAM, ~90만~180만 원) — VRAM 최강
  - RTX 5060 Ti 16GB 신품 (~84만~105만 원) — **예산 내 최선 후보**
  - RTX 3060 12GB 중고 (~35만 원) / 신품 (~59만 원) — 입문용
- USB4/TB4 대역폭과 LLM 추론 성능 분석

#### 2-2. 미니PC 방안 (독립 시스템)
- 미니PC 본체 선택지 (2026.04 기준 실제 시장가)
  - Minisforum UM890 Pro (~49만 원 16GB / ~83만 원 32GB+512GB)
  - Minisforum MS-S1 Max (128GB 통합메모리, ~350만 원 — 예산 초과 참고용)
- 미니PC + 외장 GPU 조합 가능성 (OCuLink 포트 활용)
- 미니PC 단독 사용 시 한계 (내장 GPU로는 소형 모델만 가능)

### 제3장. eGPU vs 미니PC 방식 장단점 심층 비교

#### 3-1. eGPU 방안 (Galaxy Book4 Pro + AOOSTAR AG03)

**장점:**
- 기존 노트북을 그대로 활용 → 추가 PC 구매 비용 없이 **GPU에 예산 집중 가능**
- AOOSTAR AG03 = PSU 내장 + USB4 + 100W PD → **케이블 1개로 올인원 연결**
- 외출 시 eGPU 분리하면 노트북 단독 사용 가능 (이동성 유지)
- 노트북의 16인치 AMOLED 디스플레이, 키보드 등 기존 입출력 장치 활용
- LLM 추론은 모델 로딩 후 GPU 내부 연산 위주 → **USB4 대역폭 병목이 크지 않음**
- AG03 가격이 ~30만 원으로 합리적 (ADT-Link+PSU와 비슷하나 완성도 높음)

**단점:**
- USB4/TB4 대역폭 제한 (40Gbps) → 데스크톱 대비 **prefill 단계에서 20~30% 성능 하락**
- 노트북 RAM 16GB 고정 → CPU 오프로드 시 여유 부족
- 오픈 프레임 디자인 → 먼지, GPU 팬 소음 노출
- 엔클로저 호환성 이슈 발생 가능 (드라이버, 절전모드 복귀 등)

## 6. 핵심 인사이트 (사전 조사 결과)

1. **AOOSTAR AG03가 기존 eGPU 엔클로저 대비 압도적 가성비** — 800W PSU 내장, USB4+OCuLink 지원, ~30만 원
2. **AG03 + RTX 5060 Ti 16GB 신품 = 약 114만~135만 원** → 예산 내에서 신품 보증 + 16GB VRAM 확보 (최종 추천)
3. **LLM 추론에서 USB4/TB4 대역폭 병목은 크지 않다** — 모델 로딩 후 GPU 내부 연산이 대부분
4. **RTX 3090 (24GB)으로 Gemma 3 27B, Qwen 32B까지 구동 가능** — 가장 넓은 모델 선택지
5. **미니PC 단독으로는 한계가 명확** — 결국 외장 GPU가 필요하면 노트북 + AG03와 비용 구조가 비슷해짐
6. **100만~150만 원 예산이면 AG03 + GPU 방안이 가장 합리적** — 기존 노트북을 활용하므로 GPU에 예산 집중
